Glosea
-----

Glosea是基于PHP5.3+Mysql的Web应用平台。


## Glosea的特点

## PHP开发特性摘要

- 基于PHP5.3；
- 可以使用匿名函数；
- 魔术函数_\_invoke(), _\_callStatic()；
- 使用命名空间；
- spl_autoload_register，类自动加载；
- PDO作为关系型数据库适配器；
- 使用const定义常量；
- 使用PHPDoc；
